1. Windows环境测试

测试命令行意义：

参数说明：

-f | -file file 指定文件路径

-t | -type [vmenu|vrouter] 指定解析类型

-p 指定是否是产品模式

-o | -out [console|html] filename 指定将内容解析到什么地方，后面可以指定输出文件名
